# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.0.x   | ✅ Supported       |

## Reporting a Vulnerability

If you discover a security vulnerability in APICue, please report it via email to **api-cue@gmail.com**.

Please do **not** disclose the vulnerability publicly until we have had a chance to address it.

We will:
- Acknowledge receipt within 48 hours
- Provide an initial assessment within 5 business days
- Release a fix as soon as possible (typically within 14 days)

## Scope

- The APICue plugin itself (distributed via JetBrains Marketplace)
- Token storage and handling
- HTTP request handling within the IDE
